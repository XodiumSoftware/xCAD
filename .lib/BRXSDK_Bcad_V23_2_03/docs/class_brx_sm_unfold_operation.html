◆ resultBlock()

UX_EXPORT_FUNCTION AcDbObjectId BrxSmUnfoldOperation::resultBlock ( )
virtual

Returns the owner block with unfolded geometry containing solids and 2D entities for annotation; call this function after successful 'run()' (even if operation created some warnings).

If not converted to 2D, the block contains solids per feature : each flange, unfolded bend and lofted bend is represented by separate solid; same block also contains the single solid, which is a result of union operation(s);

Returns
AcDbObjectId - AcDbObjectId of owner block with unfolded geometry
