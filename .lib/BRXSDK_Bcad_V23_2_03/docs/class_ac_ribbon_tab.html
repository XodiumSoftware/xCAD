Detailed Description

This class is used for creating ribbon tabs.

Ribbon tabs are added directly to Ribbon Control instance.

Member Function Documentation

◆ addPanel()

UX_EXPORT_FUNCTION AcRibbonUtils::AcResult AcRibbonTab::addPanel ( AcRibbonPanel *  pAcPanel)

Appends a panel to this tab.

The panel content can be either empty or can contain items.
Objects can be allocated on the heap or on the stack. Objects that are allocated on the stack are removed from the memory when they go out of scope. Objects allocated on the heap can be either deleted by the Caller or can be managed by the Application.

Parameters
[in]pAcPanelpanel to be added
Returns
AcRibbonUtils::eOk, or an error code otherwise
