LRESULT *  pResult 
)
protected

Default handler for TCN_SELCHANGE notification message.

This activates the current tab and displayes the associated Tab Child dialog. The default handler sets *pResult to 0 and returns FALSE.

Parameters
[in]pNMHdrPointer to a Windows NMHDR structure
[in]pResultPointer to a Windows LRESULT result status
Returns
TRUE if message was finally handled, FALSE to continue message processing

Default handler for TCN_SELCHANGING notification message.

The default handler calls OnTabChanging() for the current Tab Child dialog.

Returns
