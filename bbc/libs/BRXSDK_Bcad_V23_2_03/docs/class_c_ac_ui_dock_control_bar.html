UX_EXPORT_FUNCTION bool CAcUiDockControlBar::CanFrameworkTakeFocus ( )
protectedvirtual

This function determines, whether the CAD system can take the focus from controlbar.

Overridden function should return false, when it is intended to keep the focus; the default implementation returns false, when the mouse cursor is in controlbar area, true when cursor is outside the controlbar.

Returns
true, when framework can take focus (controlbar looses focus then)
