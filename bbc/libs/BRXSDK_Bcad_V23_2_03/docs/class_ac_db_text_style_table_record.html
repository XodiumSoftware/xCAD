◆ flagBits()

UX_EXPORT_FUNCTION Adesk::UInt8 AcDbTextStyleTableRecord::flagBits ( ) const

Returns the flag bits for this textstyle (Bit 2 and 3 of DXF 71).

The 2nd bit indicates, that the text ist drawn backwards, the 3rd bit indicates that the text is drawn upside down, the other bits are not used.

Returns
backwards and upside down bit.
  • 0 - neither backwards nor upside down
  • 2 - backwards, not upside down
  • 4 - not backwards, upside down
  • 6 - backwards and upside down
Remarks
In other CAD libraries the other bit values may be stored and returned. In BRX all other bit than bit 2 and bit 3 are ignored.
