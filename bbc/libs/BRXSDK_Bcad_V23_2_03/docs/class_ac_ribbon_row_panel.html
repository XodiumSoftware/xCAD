Detailed Description

This class is used for grouping items into rows.

When a row panel break is used, the items will be added in a new row.

◆ addItem()

UX_EXPORT_FUNCTION AcRibbonUtils::AcResult AcRibbonRowPanel::addItem ( AcRibbonItem *  pAcItem)

Appends an item to this row panel.

This function support all existing types of items.
Objects can be allocated on the heap or on the stack. Objects that are allocated on the stack are removed from the memory when they go out of scope. Objects allocated on the heap can be either deleted by the Caller or can be managed by the Application.

Parameters
[in]pAcItemitem to be added
Returns
AcRibbonUtils::eOk, or an error code otherwise
