◆ AcGiTextStyle() [1/2]

UX_EXPORT_FUNCTION AcGiTextStyle::AcGiTextStyle ( void  )

Default constructor.

Initializes the string values font name, big font name and style name to NULL. Initializes all the attributes to Adesk::kFalse. Initializes text size to 0.2, xscale to 1.0, oblique angle to 0 and tracking percent to 1.
