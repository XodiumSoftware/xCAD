◆ setEntities()

UX_EXPORT_FUNCTION Acad::ErrorStatus BrxSmJunctionCreator::setEntities ( const AcDbFullSubentPathArray &  entityIds)
virtual

Specifies the edges, faces and solids, on which junctions will be created.

the array may contain the sub-entity paths for a) hard edges : instead of hard edges junctions will be created b) bend or lofted bend faces : instead of bends junctions will be created c) flange faces : all the hard edges adjacent to given flange will be converted to junctions d) solids : all the hard edges of given solid will be converted to junctions

Parameters
[in]entityIds- array containing the sub-entities to be used
Returns
Acad::eOk if successful, error status otherwise
