Adds a new formula column to BOM.

The expression format is according to BOM Manager formula editor. The following symbols are supported for constructed expression: '+', '-', '*', '/', '^'. You can add '<>' with a column name to refer to the corresponding BOM column. NOTE: If @columnName is empty, a formula column with a default name will be created.

const ACHAR *  expression 
)
virtual

Adds a new template column to BOM.

The expression format is according to BOM Manager template editor. You can add '<>' with a column name to refer to the corresponding BOM column. You can add functions which has '$'(<operator>, <arg1>, <arg2>, ...) format (e.g., $(*,<Volume>, <Density>)). Delimeters should have the similar $(delim,"<symbol>") format (e.g., $(delim,";")). NOTE: If @columnName is empty, a template column with a default name will be created.

Returns
Acad::eOk, if the column was added successfully for this BrxBillOfMaterials
