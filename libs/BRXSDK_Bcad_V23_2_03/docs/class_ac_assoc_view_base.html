Returns the path to the source 3d entity fir the given part of its 2d projection.

Parameters
[in]entityId- the 2d entity inside VIEWBASE geometry block
[out]sourcePath- the AcDbFullSubentPath of the source entity, associated with the input 2d entity; the "subentity" part of 'sourcePath' object is initialised whenever possible
Returns
Acad::eOk if the associated source entity is successfully retrieved
Remarks
: this function only works with MCAD generated section entities;

AcDbFullSubentPathArray &  sourceEntitiesArray 
)
static

Returns an array of paths to all associated source entities, located at the point on 2d projection, sorted by distance from the camera position.

Parameters
[in]productPoint- a 3d point inside VIEWBASE geometry block generated with GENERATEASSOCVIEWS=On
[out]sourceEntitiesArray- array of entities located at the original point the "subentity" part of the AcDbFullSubentPath objects is initialised whenever possible
Returns
Acad::eOk if the associated source entities are successfully retrieved
Remarks
: this function only works with MCAD generated section entities;
